## 📘 About
This repository contains **code-alongs, exercises, and experiments** created while learning core **machine learning concepts in Python**.

The focus is on understanding how common machine learning algorithms work, how to implement them using popular libraries, and how to structure simple ML workflows.  
This is a **learning repository**, not a production-ready project.

---

## 🧠 What This Repository Covers
The material progresses through fundamental machine learning topics, including:

- Linear regression
- Logistic regression
- K-Nearest Neighbors (KNN)
- Random Forests
- Data preprocessing and repetition with pandas
- Input validation using Pydantic
- Building simple ML APIs with FastAPI
- Integrating scikit-learn models into APIs

The code is organized as **code-alongs and exercises** to support step-by-step learning.

## ▶️ How to Use
- Browse folders to follow topics by algorithm or concept
- Run scripts to experiment with models and data
- Some files are meant for execution, others for reading and understanding
- Virtual environment and dependency files are included for local setup

This repository works best as a **study reference and practice archive**.
